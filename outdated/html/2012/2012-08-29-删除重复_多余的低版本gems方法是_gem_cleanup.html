删除重复/多余的低版本gems方法是： gem cleanup

参考： http://docs.rubygems.org/read/chapter/10#page93

通过 gem server 后在浏览器查看本机都安装了什么gems。这个命令最好的是可以直接查看文档。

可以先 -d dryrun一下看看都会清楚哪些。然后再sudo执行。
