Three integers satisfying a2+b2=c2 are called Pythagorean numbers.

The following program calculates all pythagorean numbers less than a maximal number. Remark: We havae to import the math module to be able to calculate the square root of a number.

!/usr/bin/env python3

from math import sqrt n = int(input(“Maximale Zahl? “)) for a in range(1,n+1): for b in range(a,n): c_square = a2 + b2 c = int(sqrt(c_square)) if ((c_square - c**2) == 0): print(a, b, c) source: http://www.python-course.eu/python3_for_loop.php
