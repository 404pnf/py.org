The following objects are evaluated by Python as False:

numerical zero values (0, 0L, 0.0, 0.0+0.0j), the Boolean value False, empty strings, empty lists and empty tuples, empty dictionaries. plus the special value None. source : http://www.python-course.eu/python3_conditional_statements.php
