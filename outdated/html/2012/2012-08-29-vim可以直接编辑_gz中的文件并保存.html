vim可以直接编辑.gz中的文件并保存

$ vim some-archive.tar.gz If you vim a compressed file it will list all archive content, then you can pickup any of them for editing and saving. There you have the modified archive without any extra step. It supports many file types such as tar.gz, tgz, zip, etc.
