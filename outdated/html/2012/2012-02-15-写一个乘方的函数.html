写一个乘方的函数

例子：

(define (expt base n)

(if (= n 0)

1

(* base (expt base (- n 1)))))

This definition is given in Lisp. However, the same function can be written in any language that supports recursion. First, the function has to test for the exit condition, which in our case happens if n = 0. Then, our function multiplies the base times the exponent of the base to the original power minus 1. This parallels the mathematical definition of exponentiation. http://www.ehow.com/how_8794882_use-recursion-exponential-powers.html
