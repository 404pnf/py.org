很笨的方法，我一个一个递增数字：光标到数字上后按ctrl-a，在命令模式下 ctrl-x是递减 via http://www.vim.org/tips/tip.php?tip_id=287 You don’t even need to be on the number itself. Vim search automaticaly for the next number before incrementing / decrementing 光标无须在本行的数字上就可以按那个快捷键

.. and if you have Perl, you can do tons with the s command, eg:

:%!perl -e ‘s/(\d+)/$1 + 1/ge’

will increment every number by one. the “e” at the end makes the right side of the substitution an expression. I find this handy for converting data to/from one-based to zero-based.
