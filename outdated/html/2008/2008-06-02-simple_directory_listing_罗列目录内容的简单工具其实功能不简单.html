simple directory listing: 罗列目录内容的简单工具其实功能不简单

simple directory listing这个程序我以前试用过，很方便。功能不错，我特别看中支持输出rss。今天看到它在sourceforge上下载数量排名很靠前。项目网站在这里： http://simpledirectorylisting.net/

What’s Simple Directory Listing? PDF Print E-mail Simple Directory Listing is a single file php script which functions as directory listing in an apache http server. It provides many useful functions.

  • Copy/Move, rename, delete directories
  • Download files(pass through the php script)
  • Read/Edit files
  • Create a file/folder
  • Upload files
  • Set a particular folder as a “virtual root” for directory listing
  • Javascript/no javascript
  • Unicode supported
  • Thumbnail View
  • RSS

把文档翻译了。很简单，很短小的。

我没有使用它是因为我现在的项目都不想把文件存在自己的服务器上：从可靠性（比如能否存30年不动地方）和成本（流量大了我经济上能否承受）上看都不合适。

但你的项目，或者我的某个“商业项目”也许用得上它的 ：）。没有看到它能否结合amazon的s3使用。
