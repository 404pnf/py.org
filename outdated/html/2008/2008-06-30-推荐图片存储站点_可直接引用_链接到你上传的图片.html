地址在： http://bayimg.com/

今天glw让我推荐一个，因为近期imageshack.us有些问题。

这是 这个著名bt tracking站点提供的照片存储服务。

我信他们。：）

可以直接上传zip文件，它会自动解开并生成一个相册。

照片传错了是可以删除的，只要你给了并记得removal code

removal code是什么？

当你上传照片的时候，你可以流一个“删除代码”，就是removal code。

Please specify a removal code removal code

只要你记住这个密码，别太简单，如果你上传的照片不想让服务器保存了或者传错了，可以删除这些照片，但需要给出上传照片时的removal code来证明你是上传该照片的人。

建议大家写一个文本文件，里面对应着可能要删除的相册的removal code。removal code最好复杂一点，别用你邮箱和其他登录密码，用随机的，0fc0588a ea2c3d cdbf81a dcb18f50b87 随便给几个。如果你用gnu/linux,打开命令行，随便md5sum一个文件就能生成出类似随机密码。

  md5sum zhaopian.zip
