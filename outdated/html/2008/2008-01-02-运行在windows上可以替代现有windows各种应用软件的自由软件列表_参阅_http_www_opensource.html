好奇心强的人会问你怎么又用windows了。不是的。我整理一下办公室windows系统中的个人数据，发现里面有不少我的windows应用程序包：基本就是安装好的，可以通过移动硬盘复制到其他windows机器上使用的软件。都是我前些年常用的。现在，我不需要他们了。

原因：0) 我爱自由软件。我支持自由软件理念 1）基本不用windows了。现在有时使用时玩commando3和星际 2）运行在自由软件操作系统gnu/linux上的各种自由软件太好了、太多了。

至于为什么要把已有软件换成勇这些自由软件呢。我不想花时间解释了。这篇文章给自己和有这个需求的人看就是了。
