Comment by andresj: (1 day ago) You don’t need X11 forwarding to edit the files in Gedit nor Kate.

For Gedit, just click on “Places > Connect to server…”, select SSH from the list, write in the information, and click on Connect. Add a bookmark if you want, it will appear in the same list as Home folder, Pictures, etc. Double click on your files or File > Open inside Gedit.
