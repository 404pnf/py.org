See also Moving in a page for instructions on how to move from image to image in an emacs-w3m buffer. `T' Toggle displaying of inline images in this buffer and buffers created from this buffer, later on (`w3m-toggle-inline-images'). `I' View the image under point in an external viewer (`w3m-view-image'). `M-i' Save the image under point to an external file. The default name will be the original name of the image, so most of the time `M-i RET' will save the image with the right name (`w3m-save-image'). `M-[' Zoom out the image under point (`w3m-zoom-out-image'). `M-]' Zoom in the image under point (`w3m-zoom-in-image').
