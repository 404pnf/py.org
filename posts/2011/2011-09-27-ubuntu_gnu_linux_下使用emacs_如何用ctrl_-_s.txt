# ubuntu gnu linux 下使用emacs 如何用ctrl - space 切换中文输入法 而不是 触发emacs的 set mark

在命令行下尝试输入 LC\_CTYPE="zh\_CN.UTF-8" emacs 回车 如果输入法正常，那么修改 sudo gedit /etc/environment 增加 LC\_CTYPE="zh\_CN.UTF-8" 重新login一下就好了 来源： http://forum.ubuntu.org.cn/viewtopic.php?f=68&t=237362&start=0 set mark 可用 ctrl-shift-@ 来调出。 我切换输入法的频率比选取文本要多得多。因此我采用了上面给的方案。
