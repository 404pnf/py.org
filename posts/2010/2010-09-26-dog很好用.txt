为什么这个命令叫dog呢。因为有cat呀！ ：） cat是concatenate的缩写。 现在好了，gnu/linux上猫猫狗狗的命令都有了：） $ dog --links "http://www.domain.com" 直接显示这个页面上的所有链接。 man dog 后： > NAME > dog - better than cat > > > DESCRIPTION > dog writes the contents of each given file, URL, or the standard input if none are given or when a file > named '-' is given, to the standard output. It currently supports the file, http, and raw URL types. It > is designed as a compatible, but enhanced, replacement of cat(1).
