# boblog安装 迁移 填写数据库信息的文件在 data config php中

## 问题：
新安装可以通过图形界面填写，但我这个情况属于帮朋友把他站点从一台服务器换到另一台，不能再安装了吧。直接修改一下数据库信息就好。

想很简单的事情，就是找不着答案。

## 解决方法：

       find ./ |grep config

       cat /data/config.php|grep db
