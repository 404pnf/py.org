NAME
     rev — reverse lines of a file or files

SYNOPSIS
     rev [-V] [-h] [file ...]

DESCRIPTION
     The rev utility copies the specified files
     to standard output, reversing the order of
     characters in every line.  If no files are
     specified, standard input is read.
